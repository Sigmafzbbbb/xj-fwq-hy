# 星烬玩家联盟 - 我的世界网易版租赁服网站

这是一个为"星烬玩家联盟"我的世界网易版租赁服设计的简单网站。

## 网站结构

- `index.html` - 主页面
- `css/styles.css` - 样式表
- `js/script.js` - JavaScript 脚本
- `images/` - 图片目录

## 如何添加自己的图片

网站目前使用占位符来代替实际图片。要添加自己的图片，请按照以下步骤操作：

1. 将你的图片文件放入 `images` 目录中
2. 在 HTML 文件中更新图片路径

### 需要添加的图片：

1. **英雄区域背景图**：
   - 建议尺寸：1920x1080 像素
   - 如果要使用背景图而不是当前的渐变色，请修改 CSS 文件中的 `.hero` 类

2. **服务器展示图**：
   - 在 `images` 目录中添加 `server-showcase.jpg`
   - 建议尺寸：800x600 像素

3. **图库图片**：
   - 在 `images` 目录中添加 `gallery-1.jpg` 到 `gallery-6.jpg`
   - 建议尺寸：400x300 像素

## 自定义内容

你可以根据需要修改以下内容：

- 服务器名称和描述
- 服务器号码
- QQ群、QQ频道和Discord链接
- 服务器规则和特色
- 联系信息

## 技术说明

- 网站使用纯 HTML、CSS 和 JavaScript 构建，无需任何框架
- 响应式设计，适应不同屏幕尺寸
- 包含平滑滚动和复制服务器ID等交互功能

## 浏览器兼容性

网站兼容所有现代浏览器，包括：

- Chrome
- Firefox
- Safari
- Edge

## 部署说明

要部署此网站，只需将所有文件上传到你的网络服务器即可。网站不需要任何服务器端处理，可以作为静态网站托管。
